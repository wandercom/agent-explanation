How it works
A guest contacts Wander through any channel. The agent service identifies them, loads their conversation history across all channels, pulls relevant brand knowledge from the brain, and uses its tools to help. Every interaction is saved back — so the next agent on the next channel already knows everything.
Key idea
Each agent reads only the brain sections it needs. Sales reads identity, voice, audience, business, and hospitality. Customer service reads identity, voice, and hospitality. Marketing reads identity, voice, audience, business, and visual. Same brain, different slices.

The Brain
Shared knowledge, separate slices
The brain stores everything the agents need to know about the brand. Each agent reads only what's relevant. The brain learns from every conversation.
Identity
Business name, tagline, mission, core values, agent persona name Used by: all agents
Voice
Tone, writing style, personality, greeting, words to use and avoid Used by: all agents
Audience
Target audience description, buyer personas with motivations Used by: sales, marketing
Business
Property types, locations, USPs, pricing positioning, differentiators, common objections Used by: sales, marketing
Hospitality
Check-in experience, guest communication style, special touches, common guest questions Used by: sales, customer service
Visual
Visual style direction (cinematic, editorial, minimal), notes for image selection Used by: marketing
How it learns
Every conversation is saved to shared memory. When the sales agent learns a guest loves hot tubs, the CS agent sees that context if they call with an issue. When 23 guests mention "amazing views," the marketing agent uses their actual words in listings. The brain gets smarter with every interaction.
Unit overrides
The brain works at the organization level, but individual properties can override specific sections. Elk Ridge Lodge can have its own hospitality notes ("hot tub is behind the garage, not the deck") without changing the brand-wide defaults.
